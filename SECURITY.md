# Security policy

`hf-store-rs` has no supported release yet. Security fixes are applied to the
latest code on `main` until the first release policy is published.

Report vulnerabilities privately through
[GitHub Security Advisories](https://github.com/dinoml/hf-store-rs/security/advisories/new).
Do not open a public issue for credential exposure, path traversal, cache
poisoning, archive or link attacks, or validation bypasses.

Reports should include affected revisions, impact, reproduction details, and
any suggested mitigation. Please omit live access tokens and private repository
contents.
